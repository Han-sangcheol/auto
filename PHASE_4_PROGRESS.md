## 기술 스택

- **GUI 프레임워크**: PySide6
- **차트 라이브러리**: pyqtgraph (예정)
- **실시간 통신**: WebSocket
- **API 클라이언트**: requests

## 주요 특징

1. **반응형 UI**: 자동 새로고침 및 실시간 데이터 업데이트
2. **색상 코딩**: 손익, 상승률에 따른 직관적인 색상 표시
3. **사용자 친화적**: 확인 다이얼로그, 입력 검증
4. **모듈화**: 각 화면이 독립적인 모듈로 분리
5. **확장 가능**: 새로운 화면 추가가 용이
