# CleonAI 자동매매 프로그램

키움증권 Open API를 활용한 PC 기반 Python 자동매매 시스템입니다.

## 프로젝트 구조

이 저장소는 **PC 기반 자동매매 프로그램**을 포함하고 있습니다.

### 메인 프로젝트

📂 **`auto_trading/`** - 키움증권 자동매매 프로그램
- Python 기반 자동매매 시스템
- 다중 전략 조합 (이동평균, RSI, MACD)
- 실시간 리스크 관리
- 모의투자 및 실계좌 지원

### 시작하기

1. `auto_trading/` 폴더로 이동하세요
2. `README.md` 파일을 읽고 설치를 진행하세요
3. 자세한 내용은 `GETTING_STARTED.md`를 참고하세요

```bash
cd auto_trading
```
