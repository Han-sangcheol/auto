# 1부: 시작하기 전에

## 준비물 체크리스트

다음 항목들을 준비하세요:

- [ ] **Windows PC** (Windows 10/11, 64비트)
- [ ] **인터넷 연결** (설치 및 실시간 데이터 수신용)
- [ ] **키움증권 계좌** (모의투자 또는 실계좌)
- [ ] **공동인증서** (구 공인인증서)
- [ ] **약 30-40분의 시간**

## 중요한 주의사항

### 투자 리스크

1. **모의투자로 먼저 시작하세요**
   - 최소 1개월 이상 모의투자로 성과를 검증하세요
   - 다양한 시장 상황에서 테스트하세요
   - 실계좌 전환은 신중하게 결정하세요

2. **자동매매도 손실 위험이 있습니다**
   - 과거 성과가 미래 수익을 보장하지 않습니다
   - 시장 급변 시 예상치 못한 손실이 발생할 수 있습니다
   - 투자 가능한 금액의 일부만 사용하세요

### 법적 준수사항

1. **프로그램매매 신고**
   - 일정 규모 이상의 자동매매는 신고 의무가 있습니다
   - 한국거래소 규정을 확인하세요

2. **불공정거래 금지**
   - 시세 조종, 내부자 거래 등은 엄격히 금지됩니다
   - 관련 법규를 반드시 준수하세요

### 보안 주의사항

1. **계좌 정보 보호**
   - 계좌번호와 비밀번호를 타인과 공유하지 마세요
   - `.env` 파일을 절대로 Git에 업로드하지 마세요

2. **안전한 환경**
   - 신뢰할 수 있는 PC에서만 실행하세요
   - 백신 프로그램을 최신 상태로 유지하세요

# 2부: 환경 설정

이 부분에서는 프로그램 실행에 필요한 모든 환경을 준비합니다.

## 2.1. Windows 버전 확인

1. **Windows 키 + R** 을 누릅니다
2. `winver` 를 입력하고 Enter
3. Windows 10 또는 11이 표시되어야 합니다
4. 64비트인지 확인 (설정 → 시스템 → 정보)

> **참고**: 키움 Open API는 Windows 전용입니다. Mac이나 Linux에서는 실행할 수 없습니다.

## 2.2. Python 3.13 설치

### Python이 이미 설치되어 있는지 확인

1. **Windows 키 + R** → `cmd` 입력 → Enter
2. 다음 명령어 입력:
   ```
   python --version
   ```
3. `Python 3.11` 이상이 표시되면 **2.3단계로 건너뛰세요**

### Python 설치 (설치되지 않은 경우)

1. **Python 공식 홈페이지 방문**
   - 주소: https://www.python.org/downloads/
   
2. **"Download Python 3.13.x" 버튼 클릭**
   - 최신 안정 버전을 다운로드합니다

3. **설치 프로그램 실행**
   - 다운로드한 파일을 더블클릭
   - ⚠️ **중요**: 하단의 "Add Python to PATH" 체크박스를 **반드시 체크**하세요!
   - "Install Now" 클릭

4. **설치 완료 확인**
   - 새 명령 프롬프트 창을 열고
   - `python --version` 입력
   - Python 버전이 표시되면 성공

### 가상환경 개념 이해

프로그램마다 필요한 패키지 버전이 다를 수 있습니다. 가상환경을 사용하면:

- 프로젝트별로 독립된 Python 환경을 만들 수 있습니다
- 다른 프로젝트에 영향을 주지 않습니다
- 패키지 충돌을 방지합니다

> 이 프로그램은 `.venv` 폴더에 가상환경을 자동으로 생성합니다.

## 2.3. 키움증권 계좌 준비

### 모의투자 계좌 (권장)

1. **키움증권 홈페이지 접속**
   - 주소: https://www.kiwoom.com
   
2. **모의투자 신청**
   - 상단 메뉴 → 트레이딩 → 모의투자
   - "모의투자 신청" 클릭
   - 약관 동의 후 신청 완료
   
3. **모의투자 계좌번호 확인**
   - 신청 즉시 발급됩니다
   - 계좌번호는 **8**로 시작합니다 (예: 8123456789)
   - 초기 자금: 1억원 (가상 자금)

### 실계좌 (신중하게 결정)

- 키움증권 실제 계좌를 사용할 수 있습니다
- ⚠️ **경고**: 반드시 모의투자로 충분히 검증한 후 사용하세요
- 실계좌 번호는 8이 아닌 다른 숫자로 시작합니다

## 2.4. 공동인증서 준비

키움 Open API 로그인에는 공동인증서가 필요합니다.

### 이미 공동인증서가 있는 경우

- PC에 저장되어 있는지 확인하세요
- 키움증권 HTS(영웅문)로 로그인 테스트를 해보세요

### 공동인증서가 없는 경우

1. 키움증권 HTS(영웅문) 실행
2. 인증센터 → 공동인증서 발급/재발급
3. 안내에 따라 발급 진행

> **참고**: 모의투자도 공동인증서가 필요합니다.

## 2.5. 키움 Open API+ 신청 및 설치

자세한 내용은 [KIWOOM_API_SETUP.md](KIWOOM_API_SETUP.md)를 참고하세요.

### API 사용 신청 (1-3일 소요)

1. **키움증권 홈페이지 로그인**
   - 주소: https://www.kiwoom.com

2. **Open API 신청**
   - 트레이딩 → Open API
   - "이용 신청" 클릭
   - 약관 동의 및 신청서 작성
   - 신청 완료

3. **승인 대기**
   - 통상 1-3일 소요
   - 승인 완료 시 SMS/이메일로 알림

### Open API+ 프로그램 설치

신청이 승인된 후:

1. **Open API+ 다운로드**
   - 키움증권 홈페이지 → 트레이딩 → Open API
   - "Open API+ 다운로드" 클릭

2. **설치 실행**
   - 다운로드한 파일 실행
   - 안내에 따라 설치 진행
   - 기본 경로에 설치하는 것을 권장

3. **설치 확인**
   - `C:\OpenAPI\` 폴더가 생성되었는지 확인
   - KOA Studio가 설치되었는지 확인

### KOA Studio로 로그인 테스트

1. **KOA Studio 실행**
   - 시작 메뉴에서 "KOA Studio" 검색
   
2. **로그인 테스트**
   - 상단 "로그인" 버튼 클릭
   - 공동인증서 선택
   - 비밀번호 입력
   - 모의투자/실계좌 선택
   
3. **로그인 성공 확인**
   - 계좌번호가 표시되면 성공
   - ⚠️ 로그인이 실패하면 다음 단계로 진행할 수 없습니다

> **로그인 실패 시**: [TROUBLESHOOTING.md](TROUBLESHOOTING.md) 참고

축하합니다! 환경 설정이 완료되었습니다.

# 3부: 프로그램 설치

이제 자동매매 프로그램을 설치합니다.

## 3.1. 프로젝트 다운로드

### Git을 사용하는 경우

```bash
git clone <저장소 URL>
cd cleonAI/auto_trading
```

### Git이 없는 경우

1. GitHub 저장소 페이지에서 "Code" → "Download ZIP" 클릭
2. 다운로드한 ZIP 파일을 원하는 위치에 압축 해제
3. `cleonAI\auto_trading` 폴더로 이동

## 3.2. 설치 스크립트 실행

1. **`auto_trading` 폴더 열기**
   - Windows 탐색기에서 `auto_trading` 폴더 열기

2. **`setup.bat` 파일 더블클릭**
   - 자동으로 다음 작업이 수행됩니다:
     - Python 버전 확인
     - 가상환경 생성 (`.venv` 폴더)
     - pip 업그레이드
     - 필요한 패키지 설치 (약 3-5분 소요)
     - `logs` 폴더 생성
     - `.env` 파일 생성

3. **설치 완료 확인**
   ```
   ==========================================
       설치가 완료되었습니다!
   ==========================================
   ```
   
   이 메시지가 표시되면 성공입니다.

### 설치 중 오류가 발생한 경우

**"Python이 설치되지 않았습니다" 오류**
- Python 설치를 다시 확인하세요
- Python을 PATH에 추가했는지 확인하세요

**패키지 설치 오류**
- 인터넷 연결을 확인하세요
- 방화벽이 pip를 차단하지 않는지 확인하세요

자세한 내용은 [TROUBLESHOOTING.md](TROUBLESHOOTING.md)를 참고하세요.

## 3.3. 환경 변수 설정 (.env 파일)

프로그램 설정을 위해 `.env` 파일을 편집합니다.

### .env 파일 열기

1. `auto_trading` 폴더에서 `.env` 파일 찾기
2. 메모장 또는 텍스트 편집기로 열기
   - 파일 우클릭 → 연결 프로그램 → 메모장

### 필수 설정 항목

다음 항목들을 **반드시** 설정해야 합니다:

```env
# ==========================================
# 필수 설정 (반드시 수정하세요!)
# ==========================================

# 키움증권 계좌 정보
KIWOOM_ACCOUNT_NUMBER=8123456789          # 본인의 계좌번호로 변경
KIWOOM_ACCOUNT_PASSWORD=1234              # 계좌 비밀번호 (4자리)

# 거래 모드
USE_SIMULATION=True                       # 모의투자: True, 실계좌: False

# 관심 종목 (종목코드 6자리, 쉼표로 구분)
WATCH_LIST=005930,000660,035720           # 삼성전자, SK하이닉스, 카카오
```

### 설정 항목 설명

#### 1. 계좌 정보

```env
KIWOOM_ACCOUNT_NUMBER=8123456789
```
- 본인의 키움증권 계좌번호를 입력하세요
- 모의투자 계좌는 **8**로 시작합니다
- 실계좌는 다른 숫자로 시작합니다

```env
KIWOOM_ACCOUNT_PASSWORD=1234
```
- 계좌 비밀번호 4자리를 입력하세요
- 로그인 비밀번호가 아닙니다!
- HTS에서 주문할 때 입력하는 비밀번호입니다

#### 2. 거래 모드

```env
USE_SIMULATION=True
```
- `True`: 모의투자 모드 (권장)
- `False`: 실계좌 모드 (신중하게 사용)

#### 3. 관심 종목

```env
WATCH_LIST=005930,000660,035720
```
- 자동매매할 종목 코드를 입력하세요
- 6자리 종목코드를 쉼표로 구분합니다
- 공백 없이 입력하세요

**종목코드 찾는 방법**:
1. 네이버 증권(https://finance.naver.com) 접속
2. 원하는 종목 검색 (예: "삼성전자")
3. URL에서 숫자 확인 (예: `.../item/main.naver?code=005930`)
4. 005930이 삼성전자의 종목코드입니다

**추천 종목 (예시)**:
- 005930: 삼성전자
- 000660: SK하이닉스
- 035720: 카카오
- 051910: LG화학
- 006400: 삼성SDI

> **주의**: 너무 많은 종목을 설정하면 API 호출 제한에 걸릴 수 있습니다. 3-5개 정도를 권장합니다.

### 고급 설정 (선택사항)

필요시 다음 항목들도 조정할 수 있습니다:

```env
# 리스크 관리
MAX_STOCKS=3                              # 최대 동시 보유 종목 수
POSITION_SIZE_PERCENT=10                  # 종목당 투자 비율 (%)
STOP_LOSS_PERCENT=5                       # 손절매 비율 (%)
TAKE_PROFIT_PERCENT=10                    # 익절매 비율 (%)
DAILY_LOSS_LIMIT_PERCENT=3                # 일일 손실 한도 (%)

# 전략 파라미터
MA_SHORT_PERIOD=5                         # 단기 이동평균 기간
MA_LONG_PERIOD=20                         # 장기 이동평균 기간
RSI_PERIOD=14                             # RSI 계산 기간
RSI_OVERSOLD=30                           # RSI 과매도 기준
RSI_OVERBOUGHT=70                         # RSI 과매수 기준
MIN_SIGNAL_STRENGTH=2                     # 최소 신호 강도 (1-3)
```

### .env 파일 저장

1. 수정이 완료되면 **저장**하세요 (Ctrl + S)
2. 메모장을 닫습니다

> ⚠️ **보안 주의**: `.env` 파일을 절대로 타인과 공유하거나 Git에 업로드하지 마세요!

## 3.4. 설정 확인

설정이 올바른지 확인합니다.

1. **명령 프롬프트 열기**
   - `auto_trading` 폴더에서 주소창에 `cmd` 입력 → Enter

2. **가상환경 활성화**
   ```
   .venv\Scripts\activate
   ```
   - 프롬프트 앞에 `(.venv)`가 표시되면 성공

3. **설정 확인 명령 실행**
   ```
   python -c "from config import Config; Config.print_config()"
   ```

4. **출력 확인**
   ```
   ============================================================
   현재 설정값
   ============================================================
   계좌번호: 8123****89
   모의투자 사용: True
   최대 보유 종목 수: 3
   종목당 투자 비율: 10.0%
   ...
   ```

설정값이 올바르게 표시되면 성공입니다!

**오류가 발생한 경우**:
- `.env` 파일의 문법 오류를 확인하세요
- 필수 항목이 모두 입력되었는지 확인하세요
- [TROUBLESHOOTING.md](TROUBLESHOOTING.md)를 참고하세요

축하합니다! 프로그램 설치가 완료되었습니다.

## 4.1. 실행 전 체크리스트

## 5.3. 전략 조정하기

성과를 개선하기 위해 설정을 조정할 수 있습니다.

### 리스크 설정 조정

`.env` 파일을 열어서 수정:

```env
# 더 보수적으로 (손실 최소화)
STOP_LOSS_PERCENT=3                # 손절매를 더 빠르게
TAKE_PROFIT_PERCENT=15             # 익절매를 더 늦게
POSITION_SIZE_PERCENT=5            # 투자 비율을 줄임

# 더 공격적으로 (수익 극대화)
STOP_LOSS_PERCENT=7                # 손절매를 더 늦게
TAKE_PROFIT_PERCENT=8              # 익절매를 더 빠르게
POSITION_SIZE_PERCENT=15           # 투자 비율을 늘림
```

### 신호 강도 조정

```env
# 더 신중하게 (신호가 적지만 정확)
MIN_SIGNAL_STRENGTH=3              # 3개 전략 모두 동의 필요

# 더 적극적으로 (신호가 많지만 위험)
MIN_SIGNAL_STRENGTH=1              # 1개 전략만 동의해도 실행
```

### 관심 종목 변경

```env
# 대형주 위주
WATCH_LIST=005930,000660,051910    # 삼성전자, SK하이닉스, LG화학

# 중소형주 포함
WATCH_LIST=035720,035420,096770    # 카카오, NAVER, SK이노베이션
```

> **중요**: 설정 변경 후 프로그램을 재시작해야 적용됩니다.
