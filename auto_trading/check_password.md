# 📋 계좌 비밀번호 확인 가이드

## 🚨 2133 에러: "계좌비밀번호를 입력 및 등록 하십시오"

이 에러는 키움 HTS에 계좌 비밀번호가 등록되지 않았거나 잘못 설정되었음을 의미합니다.

## ✅ 해결 방법

### 1단계: 키움 HTS에서 계좌 비밀번호 확인/설정

1. **키움증권 HTS (영웅문) 실행**
2. **모의투자 로그인**
3. 메뉴: **[계좌관리] → [계좌비밀번호 관리]**
4. 계좌번호: **81131103** 선택
5. **비밀번호 설정/변경**
   - 새 비밀번호: `0000`
   - 비밀번호 확인: `0000`
6. **저장**

### 2단계: .env 파일 확인

```powershell
notepad auto_trading\.env
```

다음 내용 확인:

```ini
KIWOOM_ACCOUNT_NUMBER=81131103
KIWOOM_ACCOUNT_PASSWORD=0000
```

**주의:**
- 계좌번호는 **8자리** (81131103)
- 비밀번호는 **4자리 숫자** (0000)

### 3단계: 프로그램 재실행

```powershell
cd auto_trading
python main.py
```

## 🔍 비밀번호 관련 참고사항

### 모의투자 vs 실계좌

- **모의투자**: HTS에서 별도 비밀번호 설정
- **실계좌**: 실제 계좌 비밀번호 사용

### 비밀번호 형식

- **4자리 숫자**만 가능
- 예: `0000`, `1234`

### API 저장 비밀번호

키움 API에는 "자체 저장" 기능이 있어 HTS에서 한 번 입력하면 저장됩니다.
하지만 모의투자는 별도 관리되므로 반드시 설정이 필요합니다!
