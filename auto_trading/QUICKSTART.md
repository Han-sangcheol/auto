# 🚀 빠른 시작 가이드

5분 안에 자동매매 프로그램을 실행하는 가이드

## ⚡ 빠른 시작 (초보자용)

### 1. 키움 Open API 준비 ✅

먼저 [KIWOOM_API_SETUP.md](KIWOOM_API_SETUP.md)를 참고하여:

- ✅ 키움증권 Open API+ 설치
- ✅ 모의투자 계좌 신청
- ✅ 공동인증서 준비

### 2. 프로그램 설치 🔧

프로젝트 폴더에서 다음 파일을 더블클릭:

```
setup.bat
```

이 스크립트가 자동으로:
- Python 가상환경 생성
- 필요한 패키지 설치
- .env 파일 생성

### 3. 설정 ⚙️

`auto_trading/.env` 파일을 메모장으로 열어서 수정:

```env
# 필수 항목만 수정하세요!
KIWOOM_ACCOUNT_NUMBER=8123456789    # 모의투자 계좌번호
KIWOOM_ACCOUNT_PASSWORD=1234        # 계좌 비밀번호

# 관심 종목 (원하는 종목으로 변경)
WATCH_LIST=005930,000660,035720     # 삼성전자, SK하이닉스, 카카오
```

**종목 코드 찾기**: 
- 네이버 증권(https://finance.naver.com) 검색
- 예: "삼성전자" 검색 → URL의 숫자 (005930)

### 4. 실행 🎮

프로젝트 폴더에서 다음 파일을 더블클릭:

```
start.bat
```

공동인증서 창이 나타나면 인증서 선택 후 비밀번호 입력!

### 5. 모니터링 👀

프로그램이 실행되면:

```
=============================================
자동매매가 시작됩니다.
=============================================
📊 실시간 시세를 모니터링하고 매매 신호를 생성합니다.
🤖 신호 발생 시 자동으로 주문을 전송합니다.
⚠️  Ctrl+C를 눌러 언제든지 중지할 수 있습니다.
=============================================

Enter 키를 눌러 자동매매를 시작하세요...
```

**Enter** 키를 누르면 자동매매가 시작됩니다!

→ .env 파일의 계좌번호/비밀번호 확인

### 매매가 실행되지 않음
→ 로그 확인: `logs/trading.log`

→ [README.md](README.md) 참고

## ⚠️ 주의사항

1. **모의투자로 먼저 테스트!**
   - 최소 1주일 이상 모의투자로 검증
   - 성과 확인 후 실계좌 고려

2. **소액으로 시작**
   - 실계좌 전환 시 100만원 이하로 시작
   - 점진적으로 금액 증가

3. **정기 모니터링**
   - 하루 1-2번 상태 확인
   - 로그 파일 검토
   - 비정상 동작 시 즉시 중지
