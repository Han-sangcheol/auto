# ⚖️ 자동매매 프로그램 법적 제약사항 및 규정

## 📋 개요
자동매매 프로그램 개발 시 반드시 준수해야 할 한국의 법적 제약사항과 규정을 정리한 문서입니다.

## 🚨 핵심 법적 제약사항

### 1. 자본시장과 금융투자업에 관한 법률 (자본시장법)
- **제176조 (불공정거래행위의 금지)**: 부정한 방법으로 거래를 유도하는 행위 금지
- **제177조 (시세조종행위의 금지)**: 인위적인 시세 형성 목적의 매매 금지
- **제178조 (미공개정보 이용행위 등의 금지)**: 내부자 거래 금지

### 2. 프로그램매매 관련 규정
- **한국투자자보호기금 신고**: 일정 규모 이상의 프로그램매매 시 신고 의무
- **거래소 규정**: 각 거래소별 프로그램매매 규칙 준수
- **매매 주문 제한**: 과도한 주문으로 인한 시장 교란 방지

### 3. 개인정보보호법
- **제15조 (개인정보의 수집·이용)**: 개인정보 수집 시 동의 필수
- **제29조 (안전성 확보조치)**: 개인정보 암호화 및 보안 조치 의무
- **제39조의3 (개인정보 영향평가)**: 민감정보 처리 시 영향평가 수행

### 4. 정보통신망 이용촉진 및 정보보호 등에 관한 법률
- **제28조 (개인정보의 수집·이용 동의 등)**: 온라인상 개인정보 처리 규정
- **제45조 (정보보호 안전진단)**: 정보보호 관리체계 구축

## 🏢 증권사 API 이용 관련 제약

### 대신증권 CYBOS Plus API
- **개인투자자 이용 제한**: 일반 개인은 실시간 데이터 제한적 제공
- **거래량 제한**: 일일 API 호출 횟수 제한
- **계약 조건**: 별도의 API 이용 계약 필요
- **책임 면책**: API 오류로 인한 손실 책임 제한

### 기타 증권사 API
- **한국투자증권**: HTS API (개인 이용 제한적)
- **KB증권**: Open API (기관 위주)
- **미래에셋증권**: HERO API (제한적 개방)
