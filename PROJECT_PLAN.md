# 🚀 자동매매 모바일 앱 개발 프로젝트 기획서

## 📋 프로젝트 개요

### 프로젝트명
**CleonAI 자동매매 앱** (가칭)

### 목표
핸드폰 앱을 통해 대신증권 등 증권사 API를 활용하여 자동화된 주식 매수/매도 시스템 구축

### 개발 기간
약 3-6개월 (단계별 개발)

## 🎯 핵심 기능

### 1차 개발 목표 (MVP)
- [ ] 증권사 API 연동 (대신증권 CYBOS Plus)
- [ ] 기본 매매 전략 구현 (이동평균선 기반)
- [ ] 실시간 주가 모니터링
- [ ] 자동 매수/매도 기능
- [ ] 기본적인 리스크 관리 (손절/익절)

### 2차 개발 목표
- [ ] 다중 매매 전략 지원
- [ ] 백테스팅 기능
- [ ] 포트폴리오 관리
- [ ] 알림 시스템
- [ ] 상세한 거래 내역 및 수익률 분석

## ⚖️ 법적 제약사항 및 고려사항

### 중요한 법적 사항들
1. **프로그램매매 신고**: 한국투자자보호기금에 신고 필요
2. **개인정보보호법**: 사용자 데이터 보안 강화 필수
3. **자본시장법**: 불공정거래 금지 규정 준수
4. **증권사 이용약관**: 각 증권사별 API 사용 규정 준수

### 리스크 관리
- 과도한 매매 방지 장치
- 일일 손실 한도 설정
- 시장 변동성 대응 메커니즘

## 🔧 기술 스택 (예정)

### 모바일 앱 개발
- **Framework**: Flutter (크로스플랫폼)
- **Language**: Dart
- **State Management**: Riverpod/Provider

### 백엔드 서버
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: PostgreSQL
- **Cache**: Redis

### API 연동
- **증권사**: 대신증권 CYBOS Plus API
- **실시간 데이터**: WebSocket 연결

### 보안
- **인증**: JWT + OAuth 2.0
- **암호화**: AES-256
- **통신**: HTTPS/WSS

## 📱 주요 화면 구성

### 1. 로그인/인증 화면
- 증권사 계정 연동
- 생체인증 (지문, 얼굴인식)

### 2. 대시보드
- 포트폴리오 현황
- 당일 수익률
- 자동매매 상태

### 3. 전략 설정
- 매매 전략 선택/설정
- 리스크 파라미터 조정
- 백테스팅 결과 확인

### 4. 모니터링
- 실시간 주가 차트
- 매매 신호 알림
- 거래 내역

### 5. 설정
- 계정 관리
- 알림 설정
- 보안 설정

## 🔐 보안 아키텍처

### 데이터 보호
- 민감한 정보는 암호화 저장
- API 키는 서버측에서만 관리
- 통신 구간 암호화

### 인증/인가
- 다단계 인증 체계
- 세션 타임아웃 관리
- 비정상 접근 감지

## 📈 매매 전략 프레임워크

### 기본 전략들
1. **이동평균선 크로스오버**
2. **RSI 기반 과매수/과매도**
3. **볼린저 밴드 기반**
4. **MACD 신호**

### 리스크 관리
- 손절매: -5% 기본값
- 익절매: +10% 기본값
- 포지션 크기 관리
- 최대 투자 비율 제한
